# 学术写作规范（模板）

> **使用说明**：复制此文件到项目根目录并重命名为 `学术写作规范.md`，然后根据你的学校/导师要求填充内容。
> ```bash
> cp examples/学术写作规范.template.md 学术写作规范.md
> ```

---

## 一、铁律（必须遵守）

1. **去人化**：全文禁止使用"我""我们""笔者"，统一使用"本文""本章""该方案"
2. **禁用词**：<!-- 在此添加你学校/导师要求的禁用词列表 -->
   - 例：解决 → 规避/缓解/克服/应对
   - 例：所以 → 因此/鉴于此
3. **段落粒度**：每段 3-8 句
4. **句式多样性**：禁止连续两句以相同句式结构开头

## 二、章节结构模板

### 绪论结构
```
场景导入(1-2句) → 挑战(2-3句) → 方案名(1句) → 分步方案 → 验证方式(1句) → 量化效果(1句)
```

### 实验章节结构
```
引言 → 问题定义 → 方法设计 → 性能评估(理论+实验) → 本章小结
```

## 三、量化表达分级

| 提升幅度 | 推荐表述 |
|---------|---------|
| < 10% | "略有提升" |
| 10-30% | "有效提升" |
| 30-50% | "显著提升" |
| > 50% | "大幅提升" |

## 四、文献综述句式

> 禁止对所有文献使用相同的评价模板。推荐分组枚举+分号连接结构。

变体句式：
- "XX等人[N]**提出了**一种..."
- "为解决...问题，XX等人[N]**设计了**..."
- "**围绕**...需求，XX等人[N]**构建了**..."
- "**鉴于**...局限，XX等人[N]**引入了**..."
