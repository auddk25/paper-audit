# 论文审计工具设计文档

## 概述

Claude Code Skill (`/audit-paper`)，用于审计东北大学硕士学位论文。
通过 Python 脚本精确检查格式规范，Claude Code 自身完成内容审查和学术润色。

## 使用方式

```bash
/audit-paper 论文.pdf
```

## 输入文件

| 文件 | 用途 | 位置 |
|------|------|------|
| 论文 PDF | 审计对象 | 用户指定路径 |
| `5-论文格式（硕士）.pdf` | 东北大学格式规范 | 项目根目录 |
| `学术写作规范.md` | 内容审查+润色规则 | 项目根目录 |
| `基于区块链预言机...pdf` | 写作风格范例 | 项目根目录 |

## 三条审计线

### 1. 格式检查（Python + PyMuPDF）

**脚本**：`scripts/format_checker.py`

**输入**：论文 PDF 路径
**输出**：JSON 格式问题列表

**检查项**：

| 检查项 | 方法 | 规范值 |
|--------|------|--------|
| 页面尺寸 | `page.rect` | A4 (595×842pt, 即 210×297mm) |
| 版芯尺寸 | 文本区域边界计算 | 160×247mm |
| 正文字号 | `span["size"]` | 小4号 (~12pt) |
| 正文字体 | `span["font"]` | 宋体/SimSun |
| 章标题字号/字体 | 匹配"第X章" | 二号黑体 (~22pt) |
| 节标题字号/字体 | 匹配"X.X " | 三号黑体 (~16pt) |
| 款标题字号/字体 | 匹配"X.X.X " | 四号黑体 (~14pt) |
| 项标题字号/字体 | 匹配"X.X.X.X " | 小四号黑体 (~12pt) |
| 图编号连续性 | 正则匹配"图X.X" | 按章连续递增 |
| 表编号连续性 | 正则匹配"表X.X" | 按章连续递增 |
| 公式编号连续性 | 正则匹配"式(X.X)" | 按章连续递增 |
| 图题格式 | 图号下方文字 | 5号宋体 |
| 表题格式 | 表号上方文字 | 5号宋体 |
| 参考文献格式 | 正则匹配 `[N]` | 顺序编号，格式符合规范 |
| 页眉 | 提取页眉区域文本 | 左"东北大学硕士学位论文"右"章号章题" |
| 页码 | 提取页脚区域 | 阿拉伯数字居中，·N·或-N- |

**输出 JSON 格式**：

```json
{
  "issues": [
    {
      "page": 15,
      "location": "第2章标题",
      "rule": "章标题字号",
      "expected": "二号黑体 (~22pt)",
      "actual": "三号黑体 (16pt)",
      "severity": "error"
    }
  ],
  "summary": {
    "total": 12,
    "errors": 3,
    "warnings": 9
  }
}
```

### 2. 内容审查（Claude Code）

Skill 逐章读取论文 PDF，执行以下检查（基于 `学术写作规范.md` 十一节自检清单）：

**全局检查**：
- 摘要 ↔ 绪论研究内容 ↔ 总结镜像对应
- 术语一致性（跨章）
- 符号一致性（跨章）

**章节检查**：
- 逻辑连贯性：段落间是否有逻辑连接
- 论证充分性：论点是否有数据/引用支撑
- 章节衔接：过渡句是否自然
- 引用规范：图表公式是否被正文引用

**上下文传递策略**：
- 第1章分析后生成摘要（关键论点、术语表、创新点列表）
- 第N章分析时携带前N-1章摘要作为上下文
- 最后进行全局交叉检查（镜像对应、术语一致性）

### 3. 学术润色（Claude Code）

基于 `学术写作规范.md` 的表达约束，逐章提供润色建议：

- **去人化**：检测"我/我们/笔者"并建议替换
- **禁用词**：对照 7.1 替换表，标注位置并建议替换词
- **句式多样性**：检测连续相同句式开头
- **量化表达**：对照 8 节分级策略，检查表达是否匹配提升幅度
- **段落粒度**：检查段落句数是否在 3-8 句范围
- **图表引用**：检查是否先文后图、无孤立图表
- **连接词**：检查段落间是否有逻辑连接词衔接

## 技术依赖

- Python 3.10+ (Windows)
- PyMuPDF (已安装在 .venv)
- Claude Code (Skill 宿主)

## 约束与限制

- 论文限 110 页以内，每章不超过 20 页
- 格式规范硬编码为东北大学硕士论文格式
- 润色风格基于通信学报论文风格
- Claude Code Read 工具单次最多读 20 页 PDF，需分批读取
